Examples Poisson1.py to Poisson5.py contain basic examples of how to Solve
the poisson equation using underworld 3 and are designed for a new user.

Summary:
Poisson1.py: Introduction, solving the Poisson equation with a source in a Cartesian domain.
Poisson2.py: Solving the Poisson equation with a non-constant diffusivity in a Cartesian domain.
Poisson3.py: Solving the nonlinear Poisson equation in a Cartesian domain.
Poisson4.py: Non-Cartesian 2-dimensional domain example.
Poisson5.py: Non-Cartesian 3-dimensional domain example.
